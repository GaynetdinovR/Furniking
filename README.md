# Furniking
Furniking - адаптивный кроссбраузерный макет сайта
## Стек технологий

+ HTML5
+ SASS
+ GULP
+ JAVASCRIPT
+ SVG

## Установка

+ Клонировать проект

```
git clone https://github.com/GaynetdinovR/Furniking.git
```

+ Установить необходимые пакеты

```
npm install
```

+ Запустить таск-менеджер

```
gulp
```
